Tim Berners-Lee

the father of the Web

Timothy "Tim" John Berners-Lee, known as Tim Berners-Lee, is a British computer scientist, known for being the father of the World Wide Web.

It established the first communication between a client and a server using the HTTP protocol in November 1989.

In October 1994, he founded the MIT-based World Wide Web Consortium (W3C) to oversee and standardize the development of the technologies on which the Web is based and which allow the Internet to function.

Faced with the need to distribute and exchange information about his research in a more effective way, Berners-Lee developed the fundamental ideas that structure the web.

He and his group created the HTML language (HyperText Markup Language), HTTP protocol (HyperText Transfer Protocol) and URL object locating system (Uniform Resource Locator) .
